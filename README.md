# MERN Chat App

A full-stack real-time chat application built with the **MERN** stack.  

This project demonstrates hands-on learning in building a production-ready web application with authentication, messaging, and real-time updates.

---

## üöÄ Features

- User signup and login with JWT authentication  
- Real-time messaging using **Socket.IO**  
- User presence (online/offline) tracking  
- Send text and image messages  
- Responsive UI built with **React** and **Tailwind CSS**  
- Profile update functionality  

---

## üõ† Tech Stack

**Frontend**: React, Vite, Tailwind CSS, React Router, Zustand  
**Backend**: Node.js, Express, JWT Authentication, Socket.IO  
**Database**: MongoDB with Mongoose  
**Other Tools**: Git, GitHub, Cloudinary, Postman, Render
